Descarga Django

Django está disponible bajo la licencia libre BSD. Django requiere Python versión 2.4 o superior pero no depende de otras librerías de Python. Hay un par de maneras de obtener Django:

Opción 1. Obtener la última versión oficial de Django

La última versión oficial de Django es la {{ ultima_version }}. Para conseguirla descarga primero Django-{{ ultima_version }}.tar.gz. Después:

tar xzvf Django-{{ ultima_version }}.tar.gz
cd Django-{{ ultima_version }}
sudo python setup.py install

Opción 2. Obtener la última versión de desarrollo

La última y mejor versión de Django es la que está en el repositorio Subversion (sistema de control de versiones). Consíguela usando este comando de shell que requiere Git:

git clone https://github.com/django/django.git

Después de descargar Django

Lee la guía de instalación para más información.

Asegúrate de darte de alta en la lista de correo de Django en español y/o la lista de correo oficial en inglés donde otros usuarios de Django y los desarrolladores de Django se ayudan mutuamente. Lee el blog de django.es o suscríbete al feed RSS. También nos puedes seguir en Twitter.
